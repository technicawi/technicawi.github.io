Foundry is a fast, portable, and modular toolkit for Ethereum development. After installing Foundry, you have access to four tools: Tool Purpose forge Build, test, debug, deploy, and verify smart contracts cast Interact with contracts, send transactions, and query chain data anvil Run a local Ethereum node with forking capabilities chisel Solidity REPL for rapid prototyping

installing: curl -L https://foundry.paradigm.xyz | bash


Docs


Foundry Core Commands

1. forge

Compile contracts:

forge build

Run tests:

forge test

Deploy contract:


forge create src/MyContract.sol:MyContract \
  --rpc-url $RPC_URL \
  --private-key $PRIVATE_KEY

2. cast

Read block number:

cast block-number --rpc-url $RPC_URL

Call smart contract function:


cast call $CONTRACT_ADDRESS \
  "balanceOf(address)(uint256)" \
  $USER_ADDRESS

Send transaction:


cast send $CONTRACT_ADDRESS \
  "transfer(address,uint256)" \
  $TO 1000000000000000000 \
  --private-key $PRIVATE_KEY

3. anvil

Start local Ethereum node:

anvil

Fork mainnet:


anvil --fork-url $RPC_URL

4. chisel

Start Solidity REPL:

chisel

Example:


>>> uint x = 5;
>>> x * 2
